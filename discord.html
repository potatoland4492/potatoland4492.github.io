Join the Discord Server!

When you join, make sure to read #rules. If you agree, type "I agree to the Server Rules" in #rule-agreement. If you do not agree, type "Proposal to change rules" instead.
If you really have a problem, DM me or ask me to create a private thread. I am happy to try to work something out! I am always open to advice, suggestions, and moderation help.
I do my best to frequently update roles for all users. I give out roles if I know a user fits a role requirement (i.e. they play Among Us, so they get the @Among Us Players role), or they deserve a higher role (i.e. I know a user has no infractions and is helping ot keep the server organized, so I would promote them from @Knights to @Lords). In the case that you would like to have a role that I have not given you yet, ask for it in #role-request. These roles would usually be @Streamers, @Coders, @Knights, @Lords, and @Auditors. If I have repeatedly neglected giving you a role, there is probably a reason, and you may DM me or ask me to create a private thread to talk about it.
#petitions-and-request is always open if you want a bot added or any other changes made.
